# feat-tickets-split-pane

## Metadata
- ID: feat-tickets-split-pane
- Group: Content And Prompts
- Type: feature

## Summary

Show the tickets list and detail view in a side-by-side split-pane layout.

## Description

Splits the terminal horizontally: ticket list on the left (40%), detail view
on the right (60%). Depends on feat-tickets-list and feat-tickets-detail-view.
